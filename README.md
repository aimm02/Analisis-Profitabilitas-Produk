# Analisis-Profitabilitas-Produk

Proyek ini bertujuan untuk melakukan audit data menyeluruh terhadap kinerja inventaris sebuah toko ritel guna mengidentifikasi kebocoran profit dan inefisiensi pengelolaan stok. Menggunakan Python sebagai perangkat analisis utama, proses dimulai dengan pembersihan data mentah untuk mengatasi anomali kritis seperti stok negatif dan kekosongan data biaya yang dapat mendistorsi laporan keuangan. Melalui tahap feature engineering, analisis ini menciptakan metrik finansial baru berupa perhitungan margin keuntungan per unit serta valuasi aset gudang untuk memetakan kesehatan bisnis secara akurat.

Hasil eksplorasi data mengungkapkan fakta strategis bahwa mayoritas inventaris toko didominasi oleh produk dengan margin rendah, sementara kategori tertentu seperti Tembakau Original justru menyimpan potensi keuntungan terbesar namun belum dimaksimalkan ketersediaannya. Temuan ini divisualisasikan melalui dashboard interaktif yang menggabungkan analisis komposisi produk dan pemetaan kuadran aset, yang secara jelas memisahkan antara produk cash cow yang sangat menguntungkan dengan produk dead stock yang hanya membebani modal kerja. Keseluruhan analisis ini memberikan landasan berbasis data bagi manajemen untuk melakukan restrukturisasi stok, mengurangi penumpukan barang yang tidak produktif, serta menyusun ulang strategi penetapan harga demi mencapai efisiensi operasional dan lonjakan profitabilitas yang berkelanjutan.
